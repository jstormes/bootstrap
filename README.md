# bootstrap
ZF2 Module for better support of Twitter Bootstrap

## Installation via Composer

### Steps 

#### 1. Add to composer.

```
    "require" : {
        "jstormes/bootstrap" : "0.*"
    }
```


#### 2. Add module to application config (/config/application.config.php)

```PHP
   //...
   'modules' => array(
        'JStormes\Bootstrap',
   ),
   //...
```
